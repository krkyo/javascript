1. 변수 1개에 다양한 형태의 값을 저장할수 있는것이 객체 (연관된 데이터와 함수(메소드)를 저장할 수 있음)
  2. 배열선언은 [ ... ] 대괄호, 객체 선언은 {...} 중괄호
  3. 배열과 객체의 차이점, key 값의 다양성과 value값의 다양성
  4. 배열은 key가 오로지, index는(0,1,2,3,4) 숫자 자동 key가 생성
  5. 객체는 key가 사용자 정의 key로 생성(문자)
  6. 객체의 값은, 값, 메소드, 배열 모두 저장할 수 있음
  7. 객체안의 함수는 "메소드" 라고 한다.
  1. 리터럴 객체는 프로퍼티로 정의된다. (key : value) = 1개의 프로퍼티
  2. 프로퍼티 접근 : object.프로퍼티명 or object['프로퍼티명']
  3. 프로퍼티 갱신 : object.프로퍼티명 = 값
  4. 프로퍼티 삭제 : delete object.프로퍼티명
  5. 배열은 key가 오로지, index는(0,1,2,3,4) 숫자 자동 key가 생성
  6. 객체는 key가 사용자 정의 key로 생성(문자)
  7. 객체의 값은, 값, 메소드, 배열 모두 저장할 수 있음
